Manual Install

Angular Base Apps can be installed through bower or npm.

bower install angular-base-apps --save npm install angular-base-apps --save

The package includes these folders:

  • dist: compiled CSS and JavaScript files, in minified and unminified flavors.
  • js: the Angular components, and a few external libraries.
  • scss: the Sass files for each component.

Migration from 1.x to 2.x

Angular Base Apps 2.0 marks the first version of the framework to be completely separated from ZURB's Foundation for Apps. Check the migration section for complete info.

CDN

The folks at jsDeliver are hosting the Angular Base Apps CSS and JavaScript on their CDN. Just add the files as <link> or <script> tags in the <head> of your HTML file.
