Migration

Easily convert your Foundation for Apps projects to use Angular Base Apps.

Migration from Foundation for Apps

You have two options when migrating your Foundation for Apps projects over to use Angular Base Apps. You can choose use the 1.2.6 version of Angular Base Apps which provides a drop-in replacement for Foundation for Apps or you can choose to upgrade to Angular Base Apps 2.0 and prepare for the future!

In order to try out the drop-in replacement for Foundation for Apps, you can use the following CDN links to get the required files. These files should replace your foundation files with similar names.

Migration from 1.x to 2.x

Whether migrating from Foundation for Apps 1.x or Angular Base Apps 1.x, you can use the following steps to migrate to Angular Base Apps 2.x.

Rename Directive Prefix

The prefix zf used in all angular directives has been renamed to ba. Luckily zf is such a unique string that a simple search and replace can be used to make this migration easier.

Rename Angular Modules

The following angular modules have been renamed:

  • foundation → base
  • foundation.accordion → base.accordion
  • foundation.actionsheet → base.actionsheet
  • foundation.common → base.common
  • foundation.core → base.core
  • foundation.iconic → base.iconic
  • foundation.interchange → base.interchange
  • foundation.mediaquery → base.mediaquery
  • foundation.modal → base.modal
  • foundation.notification → base.notification
  • foundation.offcanvas → base.offcanvas
  • foundation.panel → base.panel
  • foundation.popup → base.popup
  • foundation.tabs → base.tabs

Rename Angular Services

The following angular services have been renamed:

  • FoundationActionSheet → BaseAppsActionSheet
  • FoundationAdapter → BaseAppsAdapter
  • FoundationAnimation → BaseAppsAnimation
  • FoundationApi → BaseAppsApi
  • FoundationModal → BaseAppsModal
  • FoundationMQ → BaseAppsMediaQuery
  • FoundationMQInit → BaseAppsMediaQueryInit
  • FoundationNotification → BaseAppsNotification
  • FoundationOffcanvas → BaseAppsOffcanvas
  • FoundationPanel → BaseAppsPanel
  • FoundationPopup → BaseAppsPopup
  • FoundationTabs → BaseAppsTabs

Rename SCSS Variables

The following SCSS variables have been renamed:

  • $foundation-colors → $base-apps-colors
  • $iconic-primary-fill → $icon-primary-fill
  • $iconic-primary-stroke → $icon-primary-stroke
  • $iconic-accent-fill → $icon-accent-fill
  • $iconic-accent-stroke → $icon-accent-stroke

Install angular-icons (optional)

If your application makes use of icons, you'll need to install angular-icons.

Install base-apps-router (optional)

If your application makes use of front matter to define states, you'll need to install base-apps-router.

Install angular-dynamic-routing (optional)

If your application makes use of front matter to define states, you'll need to install angular-dynamic-routing.

Once installed, you'll need to rename the following angular modules:

  • foundation.dynamicRouting → dynamicRouting
  • foundation.dynamicRouting.animations → dynamicRouting.animations
  • base.dynamicRouting → dynamicRouting
  • base.dynamicRouting.animations → dynamicRouting.animations
